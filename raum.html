The structure I was motivated by, Merz Akademie, had been used as a military hospital before World War II, but since 1985, it has been used as Akademie.
In one building, we can find not only the structure but also memories. Thus, this work captures each of those moments by using the alphabet characters and convey them typographically.
Following it, the work emphasizes not only inconsistency but also harmony between hospital and akademie. Moreover, it contains two books, but you can see one book outwardly.

* '48.79002788401665, 9.211585563487013' is a location number of the structure.
